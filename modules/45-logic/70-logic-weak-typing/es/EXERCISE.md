
Escribe una función `isFalsy()` que verifique si el valor pasado se interpreta como falso en PHP. Para realizar esta verificación, deberás comparar `false` con el valor pasado utilizando `==`.

```php
<?php

isFalsy(false);  // true
isFalsy(0);      // true
isFalsy('ayuda'); // false
isFalsy(10);     // false
```
